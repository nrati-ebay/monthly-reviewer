# monthly-reviewer

A web app for generating monthly team progress reports.

## Overview

Users select options from four dropdowns:

- **Month** — the reporting month
- **Year** — the reporting year
- **Team** — e.g. CLIP, CORE, ROCA, Buyer Engagement
- **Format** — PPT or PDF

The app generates a report that includes:

- JIRA ticket activity and status
- Backlog breakdown
- Sprint and velocity reports
- New Relic health statistics
